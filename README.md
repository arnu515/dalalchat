# dalalchat

Chat app made using react, express and socket.io

### Installation

1. Clone the repository:

```shell
git clone https://github.com/arnu515/dalalchat
cd dalalchat
```

2. Build the app

```shell
cd frontend
npm run build
mv build ../backend
```

3. Run the backend

```shell
export PORT=3000
npm start
```

_Use `set` instead of `export` for Windows_
